博客搭建踩坑记录

生成博客文件

这里没什么坑的，直接生成静态文件，然后进pubilc文件将生成的静态文件上传到github

我这里已经上传过一次了，第一次需要全部上传（github仓库配置请自己百度）

github pages配置


选择setting，拉到下面看见github pages项

点击开启，然后就是选取分支和pages目录了，分支没什么说的，我用的master，另可以用/docs做主目录

到这里算是弄完了，不过绑定自己的域名是有一些小坑

一些坑

1.CNMAE文件

自定义域名的CANME文件，完全不需要自己写，直接在选项里配置就是了

2.自定义域名的坑

github给的二级域名会自动跳转绑定的域名，如果绑定域名没有改域名dns，github的二级域名也会变成无法访问状态，所以最好改了域名dns记录在绑定域名
